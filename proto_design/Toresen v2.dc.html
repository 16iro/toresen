GitHub 계정을 연결해 주세요

Toresen은 여러분이 올린 PR을 읽고 리뷰해요. 그래서 GitHub 계정 연결이 필요합니다.

PR(Pull Request)이 뭔지 몰라도 괜찮아요. 내가 바꾼 코드 묶음을 트레이너에게 보여주는 통로예요. 연결만 해두면 다음 단계에서 차근차근 알려드릴게요.

내 컴퓨터와 연결하기

CLI는 ‘명령어로 컴퓨터에 말을 거는 창구’예요. 아래 한 줄을 터미널에 붙여넣으면 Toresen과 연결됩니다.

npx toresen connect --token tk_live_••••

토큰은 ‘Toresen이 나를 알아보는 비밀 열쇠’예요. 위 명령어에 자동으로 들어가 있어요. 다른 사람에게는 보여주지 마세요.
