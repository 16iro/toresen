GitHub 계정을 연결해 주세요

Toresen은 여러분이 올린 PR을 읽고 리뷰해요. 그래서 GitHub 계정 연결이 필요합니다.

PR(Pull Request)이 뭐지 몰라도 괜찮아요. 내가 바꾼 코드 묶음을 트레이너에게 보여주는 통로예요. 강의실(레포)은 둘러보기에서 마음에 드는 트랙을 포크하면 자동으로 만들어져요.

트레이너 말투를 골라보세요

리뷰를 어떤 말투로 받을지 정해요. 나중에 트레이너 탭에서 언제든 바꿀 수 있어요.

말투만 다르고, 깐깐함은 동일해요. 지적하는 내용·근거·코드 인용은 어떤 말투를 골라도 똑같이 엄격합니다. 통과 기준은 말투로 느슨해지지 않아요.
